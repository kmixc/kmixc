## About The Project

![Website Screenshot][product-screenshot]

This repository contains the source code for the **Kmixc Visuals website**, a modern, performance focused site designed to showcase creative work, services, and selected client projects.

The site is built with a strong emphasis on:
- Clean visual presentation
- Motion driven interactions
- Responsive layouts across all devices
- Fast load times and SEO friendly structure

It serves both as a portfolio and as a point of contact for potential clients looking for video production, photography, and digital media services.

## Built With

This website is built using modern frontend tools and libraries:

- **React** for component based UI
- **Vite** for fast development and optimized builds
- **GSAP** for animations and transitions
- **React Router** for navigation
- **CSS** for custom styling and layout

## Features

- Fully responsive layout for desktop, tablet, and mobile
- Animated navigation with GSAP
- Video focused hero and project sections
